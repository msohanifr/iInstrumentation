Visual Inspection

1.1 Obtain work authorisation

Has a work authorisation been obtained?

1.2 Visual inspection of PCS components (sensor, I/O board and isolation amplifier)

Has a work authorisation been obtained?

Are the loop elements clean and without any mechanical faults?

Is the loop in perfect condition?

1.3 Visual inspection in the BPCS

Is the sensor reading plausible and are there no error messages?

Check with the operator whether the current sensor switching point complies with the documentation.

1.4 Prerequisites for functional test

Bridge SIS tripping functions.

Bridge BPCS tripping functions.

Switch control to manual and/or shut down cascade.

Functional test

2.1 Check sensor safety parameters

Do the sensor safety parameters (via HART) meet the requirements?

2.2 Check measuring range

Is the sensor reading (via HART) plausible and are there no error messages?

Record calibrated measuring range and sensor unit:

Does the sensor measuring range correspond to the measuring range in the SIS operator window?

2.3 Check measuring circuit via HART

Acknowledge all BPCS/SIS alarms

Deactivate sensor write-protect

Simulate the following functions in the electronic systems of the sensor via HART:

Short-circuit simulation: HART simulation higher than 21mA.

Are there a BPCS report and an audio alarm?

Acknowledge all BPCS/SIS alarms.

Wire-break simulation: HART simulation lower than 3.6mA

Is there a BPCS report and an audio alarm?

Acknowledge all BPCS/SIS alarms.

2.4 Deactivate HART simulation

Deactivate HART simulation.

Deactivate sensor write-protect.

Check via HART: Is the sensor reading plausible and are there no error messages?

2.5 Check measuring range via HART

Disconnect sensor from process

Connect pressure simulation

Cover the sensor measuring range by means of simulation and enter the readings displayed in the SIS window in the following table.

HART simulation SIS reading Tolerance Expected reading
0 % ±xx%
50 % ±xx%
100 % ±xx%

Are the readings within the acceptable range?

Remove pressure simulation and restore correct sensor condition.

Acknowledge all BPCS/SIS alarms.

2.6 Conditions after functional test

Reset bridging of SIS tripping functions.

Reset bridging of BPCS tripping functions.

Check in BPCS: Is the sensor reading plausible and are there no error messages?

Operator to restart controller and/or cascade, if applicable.
